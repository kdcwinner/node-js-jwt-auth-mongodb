# node-js-jwt-auth-mongodb
Authentication with nodejs,mongodb and JWT(JSON Web Token)

Security

JWT authentication

Role-based access (Admin, Org User)

API rate limiting (express-rate-limit)

Secure headers (helmet)

CORS protection

Tech Stack

Node.js with Express.js

MongoDB + Mongoose

JWT for authentication

CORS, rate-limiter for security


git clone <repo-url>

cd node-js-jwt-auth-mongodb

npm install

PORT=8080

MONGO_URI=mongodb://localhost:27017/node-js-jwt-auth-mongodb

JWT_SECRET=yourstrongsecret


npm start
http://localhost:8080
